# Dropstore - POS Backend

This is the backend server for the Dropstore Point of Sale (POS) system. It handles inventory management, billing, and sales analytics.

## Tech Stack
- **Node.js & Express** (Server)
- **MongoDB** (Database)
- **Mongoose** (ORM)
